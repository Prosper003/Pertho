# Pertho

Pertho is an AI-powered PDF reader for Android. Open any PDF on your device, highlight any text, and instantly get plain-language explanations, key points, or quiz questions — without leaving the page.

## Features

- Scans your device and lists all PDF files automatically
- Clean, fast PDF reading experience
- Highlight any text to access AI features
- Explain — understand any passage in simple language
- Key Points — extract what matters most
- Quiz Me — test yourself on what you just read
- Light and dark mode
- 5 free AI actions daily
